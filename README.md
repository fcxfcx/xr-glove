# xr-glove
XR力反馈手套（大创）

## 项目规范

### 分支规范

本项目采用master-dev-feature-issue模式：
- master：主分支，不用于开发
- dev：开发主分支
- feature：特性开发分支
- issue：debug分支

其中master分支不用于开发，feature分支复制自dev分支，遵循feature-xxx命名规范。

### 提交规范
修改无论大小，都需要在commit中书写具有有意义的备注，中英文均可。

推荐的commit格式：

<对本次修改的总结> 换行
1. <具体完成的内容>
2. <具体完成的内容>
3. <具体完成的内容>
